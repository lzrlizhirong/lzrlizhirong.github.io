说明：本地项目分为两种情况：

  1. 自己本地创建的项目，从未与github产生关联；
  2. 从github上clone到本地的；

若是第1种情况，直接按下面的步骤操作即可；
若是第2种情况，又分两种情况：

  1. 若想要保留之前的commit记录，可直接修改项目根目录下的.git/config 文件，将下图中的url改为自己的github仓库地址即可，然后直接 add、commit、push 即可；
  2. 若不想保留之前的commit记录，首先要把本地项目根目录下的 .git 文件夹删掉，然后再按下面的步骤操作。
